# learning_python

### Snowflake

1. change dir to `snowflake`

```
cd snowflake
```

2. install module snowflake using venv

```
python3 -m venv venv
source venv/bin/activate

pip install snowflake-connector-python
```

3. run

```
python3 main.py
```

4. if want to deactive venv

```
deactivate
```
